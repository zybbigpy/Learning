1. 压缩解压常用

tar -cvf name.tar ./*   # 整合当前文件夹作为tar包
tar -xvf name.tar       # 解整合tar包至当前位置

tar -czvf name.tar ./*  # 创建tar.gz
tar -xzvf name.tar      # 解压tar.gz

2. 远程登陆与传送获取文件

# 远程登录
ssh usrname@ipaddr
# 向远程传送文件
scp name.tar username@ipaddr:/mwq/working_directory
# 获取远程文件
scp TOOD:
# 通过url获得文件
wget [url]
